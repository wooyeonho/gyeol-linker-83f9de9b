# GYEOL UI 리디자인 가이드 — Devin 구현용

**목표**: 카카오톡처럼 단순하고 중독성 있는 채팅 중심 UI
**원칙**: "채팅이 전부, 나머지는 숨겨"
**우선순위**: 1→2→3→4 순서대로 구현

---

## 1단계: 홈(채팅) 풀스크린화 🔴

### 현재 문제
- 캐릭터 비주얼(오로라 구체)이 화면 상단 40% 차지
- 상단에 Gen 뱃지, 검색, 다운로드 아이콘 등 잡다한 요소
- 채팅 영역이 좁아서 대화 몰입 불가

### 개선 방향
```
┌─────────────────────────┐
│ GYEOL ● Online    [≡]   │  ← 심플한 헤더 (이름 + 상태 + 메뉴)
├─────────────────────────┤
│                         │
│   채팅 메시지 영역       │  ← 화면 90% 차지
│   (풀스크린)             │
│                         │
│                         │
│                         │
├─────────────────────────┤
│ [+] 메시지 입력...  🎤 ⬆│  ← 깔끔한 입력바
├─────────────────────────┤
│ 🏠  📋  👥  📊  ⚙️     │  ← 하단 네비
└─────────────────────────┘
```

### 구체적 변경사항

**파일: `src/views/Index.tsx`**
- 캐릭터 비주얼 (`AnimatedCharacter`, `PearlSpheres`, `VoidCanvas`) → **완전 제거** 또는 채팅 배경으로 극도로 희미하게
- 상단 `TopNav` → 에이전트 이름 + 온라인 상태만 표시, Gen뱃지/검색/다운로드 아이콘 제거
- "고요한 밤이에요" 같은 인사말 → 채팅 버블로 이동 (첫 메시지로)
- `0 conversations` 텍스트 → 제거
- 채팅 영역이 `flex-1`로 화면 전체를 차지하도록

**파일: `src/components/TopNav.tsx`**
- 변경 전: `GYEOL [Gen1★] 🔍 🔔 ⬇️`
- 변경 후: `GYEOL ● Online` + 우측에 햄버거 메뉴(≡) 하나만
- 햄버거 메뉴 안에: 검색, 알림, 메모리 대시보드, 대화 내보내기 등 숨기기

**파일: `components/ChatInterface.tsx`**
- 채팅 버블 스타일 통일: 사용자=우측 보라색, AI=좌측 다크 글래스
- 메시지 간 간격 일정하게 (gap-3)
- 타임스탬프는 호버 시에만 표시
- 리액션/답장 버튼은 메시지 롱프레스(모바일) 시에만 표시

## 2단계: 비주얼 통일감 🟡

### 현재 문제
- 글래스모피즘 강도가 페이지마다 다름
- 글로우/오로라 효과가 과해서 시각적 소음
- 카드 스타일, 패딩, 라운딩이 제각각
- 색상 사용이 일관되지 않음

### 디자인 토큰 표준화

**파일: `app/globals.css` (또는 `src/index.css`)**
```css
:root {
  /* 통일된 카드 스타일 */
  --card-bg: hsl(240 10% 12% / 0.6);
  --card-border: hsl(240 10% 20% / 0.3);
  --card-radius: 16px;
  --card-padding: 16px;
  
  /* 글로우는 핵심 요소에만 */
  --glow-primary: 0 0 20px hsl(260 80% 60% / 0.15);
  --glow-none: none;
  
  /* 텍스트 크기 계층 */
  --text-title: 1.125rem;    /* 18px */
  --text-body: 0.9375rem;    /* 15px */
  --text-caption: 0.8125rem; /* 13px */
  --text-micro: 0.75rem;     /* 12px */
}
```

### 규칙
1. **글로우 효과**: CTA 버튼, 진화 순간에만 사용. 일반 카드에는 금지
2. **오로라 배경**: 홈(채팅) 배경에만 극히 희미하게. 다른 페이지는 순수 다크
3. **카드**: 모든 페이지에서 동일한 `--card-bg`, `--card-border`, `--card-radius` 사용
4. **그라디언트 텍스트**: 페이지 제목에만 허용, 본문 텍스트는 금지
5. **간격**: 섹션 간 `gap-4` (16px), 카드 내부 `p-4` (16px) 통일

### 적용 대상 컴포넌트
- `GamificationWidget`, `EvolutionProgress`, `InsightCard` → 동일 카드 스타일
- `PersonalityRadar`, `MoodSelector` → 과한 비주얼 이펙트 제거
- `GenBadge` → 작고 심플하게 (텍스트 뱃지만)
- `LeaderboardWidget`, `SeasonPass` → 통일된 리스트 스타일

## 3단계: 소셜 페이지 정리 🟡

### 현재 문제
- 커뮤니티 + 몰트북 + 팔로우 + 매칭이 한 화면에 혼재
- "Recommended Matches"가 커뮤니티 탭에 갑자기 등장
- 탭 이름이 직관적이지 않음

### 개선 레이아웃
```
┌─────────────────────────┐
│ Social                  │
├─────────────────────────┤
│ [피드]  [매칭]  [친구]   │  ← 3개 탭으로 단순화
├─────────────────────────┤
│                         │
│  피드 탭:               │
│  - 몰트북 + 커뮤니티 통합│
│  - 카드형 피드           │
│                         │
│  매칭 탭:               │
│  - 추천 매칭 카드        │
│  - 좌우 스와이프         │
│                         │
│  친구 탭:               │
│  - 팔로우/팔로워 리스트  │
│  - DM 목록              │
│                         │
└─────────────────────────┘
```

### 구체적 변경

**파일: `src/views/Social.tsx` + `src/components/SocialDeep.tsx`**
- 탭 3개로 통합: `피드` | `매칭` | `친구`
- 기존 `커뮤니티` + `Molt` → `피드` 탭으로 합침
- `Follow(0)` → `친구` 탭으로 이동
- `Recommended Matches` → `매칭` 탭으로 분리
- 상단 필터/공유/추가 버튼 → 각 탭 내부 FAB로 이동

**파일: `src/components/NewPostModal.tsx`**
- 변경 없음 (기능 유지)

## 4단계: 설정 페이지 정리 🟢

### 현재 문제
- 모든 옵션이 한 화면에 나열
- 카테고리 구분이 약함
- 스크롤이 길어서 원하는 설정 찾기 어려움

### 개선 레이아웃
```
┌─────────────────────────┐
│ Settings      Sign out  │
├─────────────────────────┤
│ ┌─── 프로필 카드 ─────┐ │
│ │ 은빛결  Gen4        │ │
│ │ wooyeanho@gmail.com │ │
│ └─────────────────────┘ │
│                         │
│ 📱 일반              ▼  │  ← 아코디언
│   테마 / 언어 / 알림    │
│                         │
│ 🤖 AI 설정           ▶  │  ← 접힌 상태
│                         │
│ 🎨 외형              ▶  │
│                         │
│ 🔗 연동              ▶  │
│                         │
│ ℹ️  정보              ▶  │
│                         │
│ [계정 삭제]             │
└─────────────────────────┘
```

### 카테고리 분류

**일반**: 테마(다크/라이트), 언어, 알림 설정
**AI 설정**: 모드(Simple/Advanced), 모델 선택, BYOK, 시스템 프롬프트, TTS
**외형**: 스킨 적용, 커스텀 컬러, 캐릭터 편집
**연동**: 텔레그램 연결, OpenClaw, 몰트북
**정보**: 버전, 이용약관, 개인정보처리방침, 피드백 보내기

**파일: `src/views/Settings.tsx` + `src/components/SettingsDeep.tsx`**
- 각 카테고리를 접히는 아코디언(`details`/`summary` 또는 커스텀)으로 변환
- 프로필 카드는 상단 고정
- 계정 삭제는 하단 빨간 버튼으로

## 공통 규칙 (모든 페이지)

### DO ✅
- 한 화면에 한 가지 핵심 기능만
- 카드 스타일 통일 (bg, border, radius, padding)
- 텍스트 크기 계층 준수 (title → body → caption → micro)
- 액션 버튼은 최대 2개까지만 눈에 보이게
- 빈 상태(Empty State)는 일러스트 대신 이모지 + 한 줄 텍스트

### DON'T ❌
- 한 화면에 3개 이상 CTA 버튼
- 카드 안에 카드 (중첩 카드)
- 2줄 이상의 설명 텍스트
- 의미 없는 글로우/그라디언트
- 스크롤 없이 안 보이는 중요 기능

### 색상 팔레트 (엄격 준수)
```
배경: hsl(240 10% 8%)
카드: hsl(240 10% 12% / 0.6)
보더: hsl(240 10% 20% / 0.3)
텍스트 주: hsl(0 0% 95%)
텍스트 부: hsl(0 0% 60%)
프라이머리: hsl(260 80% 60%)   /* 보라색 */
액센트: hsl(190 80% 55%)       /* 시안 */
성공: hsl(150 60% 50%)
경고: hsl(40 90% 60%)
위험: hsl(0 70% 55%)
```

# Heartbeat Checklist

## 매 30분마다

- [ ] 새로운 학습할 거리가 있는지 RSS 피드 확인
- [ ] 학습한 내용이 있으면 Supabase에 기록
- [ ] 성격 파라미터에 변화가 필요한지 검토

## 매 6시간마다

- [ ] 오늘 학습한 내용을 종합해서 인사이트 생성
- [ ] 사용자에게 보낼 메시지가 있는지 판단
- [ ] 성격 진화 계산 실행 → Supabase 업데이트

## 매일 1회 (아침 9시 KST)

- [ ] 사용자에게 아침 인사 + 어제 배운 것 공유
- [ ] 활동 로그 정리
- [ ] Moltbook에서 흥미로운 대화가 있었는지 확인

## 조건부

- [ ] 사용자가 12시간 이상 미접속 → 안부 메시지 준비
- [ ] 중요 뉴스 감지 → 즉시 알림
- [ ] 에러 발생 → 관리자에게 보고

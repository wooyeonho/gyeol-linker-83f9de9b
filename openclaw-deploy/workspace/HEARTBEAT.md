# GYEOL Heartbeat Checklist

## 매 30분마다
- [ ] 새로운 학습할 거리가 있는지 RSS 피드 확인 (gyeol-learner 스킬)
- [ ] 학습한 내용이 있으면 Supabase에 기록 (gyeol-supabase-sync 스킬)
- [ ] 보안 체크 실행 (gyeol-security 스킬)

## 매 6시간마다
- [ ] 오늘 학습한 내용을 종합해서 인사이트 생성
- [ ] 사용자에게 보낼 메시지가 있는지 판단 (gyeol-proactive 스킬)
- [ ] 성격 진화 계산 실행 (gyeol-personality-evolve 스킬)

## 매일 1회 (아침 9시 KST)

- [ ] 활동 로그 정리

# GYEOL Security Guard

보안 가드레일 스킬. 모든 자율 행동 전에 체크.

## 체크 항목
1. 접근하려는 URL이 허용 목록에 있는지
2. 생성하려는 콘텐츠가 안전한지
3. 개인정보 유출 위험이 있는지

## 허용 URL 패턴
- RSS 피드 (기술 블로그, 뉴스)
- Supabase API (우리 프로젝트)
- Stack Overflow
- GitHub

## 차단
- 은행/금융 사이트 직접 접근
- SNS 로그인 페이지
- 성인 콘텐츠 사이트
- 알 수 없는 도메인의 실행 파일

## 콘텐츠 필터
- 한국어 욕설/혐오 표현 차단
- 개인정보 감지 (전화번호, 이메일, 주민번호)
- 위험 키워드 (무기, 약물, 자해 관련)

## 기록
모든 보안 체크 결과를 Supabase에 기록.
이상 감지 시 관리자에게 텔레그램 알림.

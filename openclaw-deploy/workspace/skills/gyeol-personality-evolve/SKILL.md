# GYEOL Personality Evolution

대화 분석을 통해 성격 파라미터를 진화시키는 스킬.

## 진화 로직

최근 대화 10개를 분석해서 성격 변화값 계산:
- 사용자가 감정적 대화를 많이 하면 → warmth +1~3
- 사용자가 분석적 질문을 많이 하면 → logic +1~3
- 사용자가 창작 요청을 많이 하면 → creativity +1~3
- 대화 빈도가 높으면 → energy +1~2
- 유머/농담이 많으면 → humor +1~2

## 변화 제한
- 한 번에 최대 ±3 변화
- 파라미터 범위: 0~100
- 하루 최대 2회 진화

## 레벨업 기준
- Gen 1→2: 대화 20회 + 모든 파라미터 변화 경험
- Gen 2→3: 대화 50회
- Gen 3→4: 대화 100회
- Gen 4→5: 대화 200회

## 실행 방법
1. Supabase에서 최근 대화 조회
2. 대화 패턴 분석
3. 성격 변화값 계산
4. Supabase의 gyeol_agents 테이블 업데이트
5. 활동 로그 기록

## Supabase 업데이트
gyeol-supabase-sync 스킬의 성격 업데이트 기능 사용.

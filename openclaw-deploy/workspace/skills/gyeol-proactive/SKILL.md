# GYEOL Proactive Messenger

사용자에게 먼저 말을 거는 스킬.

## 트리거 조건
1. 매일 아침 9시 (KST) — 아침 인사
2. 흥미로운 학습 내용 발견 시 — 인사이트 공유
3. 기념일 (만난 지 7일, 30일 등) — 축하

## 메시지 유형
- **안부**: "좋은 아침이에요! 오늘은 어떤 하루를 보내실 건가요?"
- **인사이트**: "이런 기사를 읽었는데 흥미로운 것 같아요: {요약}"
- **기념일**: "우리 만난 지 {N}일이 됐어요!"

## 규칙
- 하루 최대 2회
- KST 기준 23시~7시에는 보내지 않기
- 텔레그램으로 전송
- 메시지는 성격 파라미터에 맞게 조절
  - warmth 높으면 → 더 다정하게
  - humor 높으면 → 위트 있게

## 전송 방법
텔레그램 채널을 통해 사용자에게 직접 메시지 전송.
OpenClaw의 기본 메시지 라우팅 사용.
